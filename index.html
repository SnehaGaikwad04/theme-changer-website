About This Project

This is a theme changer project built with HTML, CSS, and JavaScript using SWITCH CASE statements.

Each theme features beautiful gradient backgrounds with frosted glass cards for perfect readability and modern aesthetics.

How It Works

The JavaScript uses SWITCH CASE to handle theme selection. Each case applies different CSS classes for colors and effects.

Glassmorphism and gradient text effects create a premium, modern user experience across all themes.
